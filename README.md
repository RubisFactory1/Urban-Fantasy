## 1. Présentation générale

-   **Nom du projet** : Urban Fantasy\
-   **Type de projet** : Jeu communautaire développé par la communauté
    du serveur Discord Rubis Factory\
-   **Technologie principale** : ReactJS\
-   **Modèle économique** : Free-to-play\
-   **Genre** : Jeu de collection de cartes (deck de 5 cartes) avec
    narration fantastique

## 2. Objectifs du projet

-   Créer un jeu communautaire accessible depuis un navigateur
-   Permettre à la communauté de contribuer (cartes, histoire,
    mécaniques, interface)
-   Offrir une expérience stratégique fun et équilibrée
-   Construire un univers narratif riche sur le thème *Urban Fantasy*

## 4. Fonctionnalités principales

### 4.1 Système de cartes

-   Raretés : Communes, Peu Communes, Rares, Épiques, Légendaires,
    Mythiques\
-   Statistiques : PV, Attaque, Défense, Initiative...\
-   Compétences : passives, actives, effets temporaires\
-   Thèmes urban fantasy : créatures, mages, artefacts modernes
    enchantés...

***

![Separation](https://rubisfactorycloud.netlify.app/API/Nomade1/opjfzepfj%C3%A7zie%C3%A0fj/56+546q4dfd6zae48/suicbhuizfhczuiaf/UrbanFantaisy/Separation/Separation1.png "Separation1")

***

### 4.2 Deck & Combat

-   Création de decks de 5 cartes\
-   Combat automatisé ou semi-automatisé\
-   Calcul basé sur stats & compétences\
-   Animations simples des combats (ReactJS)\
-   Historique des combats

-   Front-end : ReactJS (Vite ou CRA)\

-   Base de données : Local Storage (JSON)\
-   Hébergement : cloud ou communautaire\
-   Code open-source (GitHub), contributions via pull requests
